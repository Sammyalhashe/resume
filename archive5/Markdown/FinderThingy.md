#

* Wrote an iOS app in Swift3 in Xcode.
* Personalized list of branches where I could look up the closest locations to me.
* App structured using MVC.
* Used [Alamofire](https://github.com/Alamofire/Alamofire) to access the Google Maps/PlacesAPI.
* De-structured the returned JSON to supply the user with direction information.
* This was an introductory app that really wasn’t that great at all, but I’m proud of the fact that it was my first.
