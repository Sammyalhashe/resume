#

* Single-page application written with Angular6 frontend.
* Material design principles are followed by theming the app with Angular Material.
* Has account authentication with login screen.
* User has the ability to login with their own personal email or their Google account through Firebase Authentication
* Using the [<code>angularfire2</code>](https://github.com/angular/angularfire2) module for integrating with Firebase's Firestore, users are able to create, delete, and edit posts.
