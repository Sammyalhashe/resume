- Characterized the Mathblock component of Microsemi's G5 FPGA product as well as performance-related AC tests on other combinatorial and sequential elements found in the FPGAs that went to market.
    - Example components include look-up tables, carry chains, digital I/O's, characterized mainly for max frequency.
    - Tested chains on different corners of the FPGA fabric.
- Inside Libero, I created Verilog test structures, making use of JTAG to read input from test programs created using a company code.
- Wrote sections of the characterization report found in the official product manual used to facilate customer knowledge in our device.
