#

- Given a single dashcam video as training data, the task was to build a model that could determine the speed of a car.
- Using openCV, I split the data into frames and performed optical flow in a pre-processing step.
- Built a convolutional network based on an NVIDIA paper for their self driving cars.
- Project is still in progress, next steps include splitting the data into testing and validation sets and tune any extra parameters.
